## Struktur Project
- /data: berisi dataset yang digunakan dalam format .csv
- /dashboard: berisikan main.py dan dataset dashboard untuk membuat dashboard 
- notebook.ipynb: digunakan untuk analisis data

## Instalasi
1. Clone repository:
   ```shell
   [git clone https://github.com/sh444nimain/Giga-newFundamentalAnalisisData.git](https://github.com/sh444nimain/Giga-newFundamentalAnalisisData.git)
   ```
2. Install dependencies
   ```shell
   pip install streamlit
   pip install -r requirements.txt
   ```
3. Jalankan aplikasi

## Penggunaan
1. Masuk ke local project
   ```shell
   cd submission/dashboard/
   streamlit run dashboard.py
   ```
